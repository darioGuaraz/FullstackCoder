1.ª Entrega - Dario Hernán Guaraz

En esta primera entrega se desarrolló una estructura web utilizando HTML y hojas de estilo en CSS. El sitio cuenta con una barra de navegación que permite recorrer las distintas secciones de forma ordenada y clara. Se aplicaron conceptos fundamentales de maquetado y diseño para lograr una presentación visual coherente y funcional.
